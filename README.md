# 🚖 Ola Ride Analytics Dashboard

## 📌 Project Overview

This Power BI dashboard analyzes ride-sharing business data and provides insights into revenue, profit, trip patterns, customer ratings, vehicle performance, and operational efficiency.

The project simulates real-world analytics used by ride-sharing companies like Ola.

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- CSV Dataset

---

## 📊 KPIs

- Total Revenue
- Total Profit
- Total Trips
- Average Fare
- Average Driver Rating
- Cancellation Percentage

---

## 📈 Dashboard Pages

### Page 1: Executive Overview
- Revenue Trend
- Vehicle-wise Revenue
- Payment Mode Distribution
- Ride Status Analysis
- City-wise Revenue Map
- Peak Hour Analysis

### Page 2: Customer and Driver Analysis
- Top Pickup Locations
- Top Drop Locations
- Driver Rating Distribution
- Customer Rating Distribution
- Distance vs Fare Analysis

### Page 3: Profit Analysis
- Weather-wise Revenue
- Vehicle-wise Profit
- Fuel Cost Analysis
- Profit Margin Analysis
